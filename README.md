This project analyzes student performance data and predicts exam scores using machine learning.

The workflow includes exploratory data analysis, data quality checks, preprocessing, train/validation/test splitting, and model training with Random Forest regression and GridSearchCV.

It was built in Python using Pandas, NumPy, Matplotlib, Seaborn, and scikit-learn to understand which academic and personal factors are linked to student results
